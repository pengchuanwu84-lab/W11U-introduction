## Featured Projects

### Job Search AI Agent

A lightweight AI Agent for resume-JD matching and interview preparation.

This project reads a resume and a job description, then generates a structured Markdown report, including job requirement analysis, resume matching evidence, resume revision suggestions, interview preparation prompts, and next-step actions.

**Tech Stack:** Python, DeepSeek API, OpenAI-compatible Chat Completions, Tool Calling, Markdown

**Repository:**  
https://github.com/pengchuanwu84-lab/job-search-ai-agent

### Multimodal Fatigue Detection and Real-time Intervention System

A prototype system for fatigue detection and real-time intervention based on EEG, vision, Raspberry Pi edge computing, and audio feedback.

The system follows the workflow of data acquisition, multimodal fusion judgment, real-time intervention, and experiment log recording.

**Tech Stack:** Python, Raspberry Pi, EEG, K230, USB Camera, PyQt, Audio Intervention

**Repository:**  
https://github.com/pengchuanwu84-lab/multimodal-fatigue-intervention-system
